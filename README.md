# Supply-Chain-Performance-Analysis
Overview

Developed an end-to-end supply chain analytics project using Excel and Tableau to evaluate inventory performance, supplier reliability, transportation efficiency, and product profitability.

Business Problem

Analyze operational data to identify opportunities for improving inventory planning, supplier performance, and logistics efficiency.

Tools:
Microsoft Excel,
Tableau Public

Analysis Included:
Revenue Analysis,
Inventory Risk Assessment,
Supplier Performance Scorecards,
Transportation Cost Analysis,
Profit Margin Analysis,
Executive KPI Dashboard

Deliverables:
Excel Analysis Workbook,
Tableau Dashboard,
Executive Summary

Key Findings:
Skincare products generated the highest revenue.
Supplier performance varied across defect rates and lead times.
High inventory risk products were identified using stock levels and lead times.
Transportation costs differed across shipping modes.
